# 💻 Projetos Pessoais - Gabriel Rios
   
Espaço exclusivo para laboratórios, testes independentes e estudos autônomos realizados por **Gabriel Rios** para expandir os conhecimentos adquiridos no **Bootcamp**.

## 🛠️ Tecnologias de Interesse para Exploração
Para além da grade curricular do curso, este espaço visa documentar o aprendizado em:
* **Ferramentas de Desenvolvimento**: Aprofundamento em Git, GitHub Actions e ambientes de desenvolvimento modernos.
* **Exploração de Novas Stacks**: Testes práticos com frameworks e bibliotecas modernas do ecossistema de software.
* **Projetos Autônomos**: Criação de ferramentas e utilitários que resolvam problemas do dia a dia para fixar conceitos.

## 💡 Filosofia de Estudo
"A melhor forma de aprender a programar é construindo." Este diretório reflete a proatividade em ir além do que é exigido em sala de aula.
